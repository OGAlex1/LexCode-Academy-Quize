# LexCode Quiz App

A dynamic, web-based quiz application that allows users to answer multiple-choice questions, track their score, and see results in real-time. Built with HTML, CSS, and JavaScript, this app features timers, interactive answer selection, and a user-friendly interface.


## Features
- Start Quiz with Name Input: Users must enter their name before starting.

- Randomized Questions and Answers: Each quiz session shuffles questions and answer options for a unique experience.

- Per-Question Timer: Each question has a countdown timer to encourage quick thinking.
Total Quiz Timer: Tracks total time for the entire quiz.

- Answer Highlighting: Correct and incorrect answers are visually marked.

- Next Question Navigation: Users can proceed after selecting an answer.

- End Quiz Button: Allows users to quit anytime and see their score.

- Results Overlay: Displays a personalized score and a congratulatory message.

- Restart Quiz: Users can restart the quiz at any time.

- Dark Mode Toggle: Optional dark mode for better accessibility.

## Language Used

- HTML5 – Structure of the quiz interface

- CSS3 – Styling, animations, responsive layout

- JavaScript (ES6) – Quiz logic, timers, DOM manipulation

### Usage
- Open index.html in your browser.

- Enter your name and click Start Quiz.

- Answer the questions within the time limit.

- Click Next to move to the following question.

- You can click End Quiz anytime to view your score.

- After finishing, a results overlay will display your score with a personalized message.

- Click Restart Quiz to play again.
